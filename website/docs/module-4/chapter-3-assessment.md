# Chapter 3 Assessment: Cognitive Planning & Capstone Overview

## Assessment Questions

### Question 1
How do Large Language Models enable hierarchical task decomposition in robotics?
- A) By generating random task sequences
- B) Through sophisticated reasoning to break down complex goals into manageable subtasks with dependency analysis
- C) By only processing simple commands
- D) By eliminating the need for task decomposition

**Answer: B**

### Question 2
What are the key components of Hierarchical Task Networks (HTNs) in cognitive planning?
- A) Only primitive tasks
- B) Primitive tasks, compound tasks, methods for decomposition, and constraints
- C) Only compound tasks
- D) Only methods for decomposition

**Answer: B**

### Question 3
What are the key steps in translating high-level goals to executable ROS 2 action sequences?
- A) Only goal identification
- B) Goal parsing, action sequence generation, and validation/refinement
- C) Only action execution
- D) Only parameter mapping

**Answer: B**

### Question 4
What is involved in goal parsing with LLMs?
- A) Only recognizing keywords
- B) Goal identification, parameter extraction, constraint recognition, and ambiguity resolution
- C) Only converting text to speech
- D) Only storing the goals

**Answer: B**

### Question 5
What are the main action categories in ROS 2 action libraries for cognitive planning?
- A) Only navigation actions
- B) Navigation, manipulation, perception, and communication actions
- C) Only manipulation actions
- D) Only communication actions

**Answer: B**

### Question 6
What validation checks are important for action sequences?
- A) Only checking if actions are grammatically correct
- B) Logical consistency checking, capability validation, and resource validation
- C) Only checking if actions are short
- D) Only checking if the robot is on

**Answer: B**

### Question 7
What are the levels of behavioral architecture in autonomous humanoid systems?
- A) Only executive level
- B) Executive, tactical, operational, and reflexive levels
- C) Only operational level
- D) Only executive and operational levels

**Answer: B**

### Question 8
What are the components of multi-modal integration in cognitive planning systems?
- A) Only perception integration
- B) Perception, language, and action integration
- C) Only language integration
- D) Only action integration

**Answer: B**

### Question 9
How does contextual reasoning enhance cognitive planning?
- A) By ignoring the environment
- B) By incorporating environmental context, adapting plans dynamically, and handling situational changes
- C) By making plans static
- D) By eliminating the need for planning

**Answer: B**

### Question 10
What are the differences between sequential and parallel planning?
- A) No differences exist
- B) Sequential planning involves linear execution with dependency chains, while parallel planning allows simultaneous execution of independent tasks
- C) Only sequential planning is used in robotics
- D) Only parallel planning is used in robotics

**Answer: B**

### Question 11
What safety mechanisms are critical for autonomous cognitive planning systems?
- A) Only fail-safe behaviors
- B) Fail-safe behaviors, constraint enforcement, human override, and risk assessment
- C) Only human override
- D) Only risk assessment

**Answer: B**

### Question 12
How does the capstone project demonstrate integration of all VLA system components?
- A) Only using perception
- B) By combining perception, language understanding, action execution, and planning in a cohesive autonomous system
- C) Only using language processing
- D) Only using action execution

**Answer: B**
