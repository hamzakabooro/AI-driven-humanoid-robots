# Chapter 3 Assessment: Navigation with Nav2 for Humanoids

## Assessment Questions

### Question 1
What are the main differences between wheeled robot and humanoid robot navigation?
- A) There are no differences
- B) Humanoid navigation must account for bipedal locomotion constraints and balance maintenance
- C) Wheeled robots are always faster
- D) Humanoid robots don't need path planning

**Answer: B**

### Question 2
How does footstep planning differ from traditional path planning?
- A) It's identical to traditional path planning
- B) It must consider discrete footholds and bipedal dynamics
- C) It's simpler than traditional path planning
- D) It doesn't consider obstacles

**Answer: B**

### Question 3
What are Zero Moment Point (ZMP) constraints and why are they important?
- A) They relate to robot speed only
- B) They ensure balance and stability during bipedal locomotion
- C) They reduce computational requirements
- D) They improve sensor accuracy

**Answer: B**

### Question 4
How can Nav2 behaviors be customized for humanoid platforms?
- A) By using the same configuration as wheeled robots
- B) Through custom costmap configurations and specialized local planners
- C) By removing safety checks
- D) By disabling obstacle avoidance

**Answer: B**

### Question 5
What is the primary purpose of Navigation2 (Nav2)?
- A) To simulate robot behavior
- B) To provide the latest navigation framework for ROS 2
- C) To manufacture robots
- D) To train AI models

**Answer: B**

### Question 6
Which of the following is a challenge specific to humanoid robot navigation?
- A) Wheel wear and tear
- B) Bipedal locomotion constraints and balance maintenance
- C) Battery charging
- D) Sensor calibration

**Answer: B**

### Question 7
What does ZMP stand for?
- A) Zero Motion Point
- B) Zero Moment Point
- C) Zero Momentum Position
- D) Zone Management Protocol

**Answer: B**

### Question 8
What is a behavior tree used for in Nav2?
- A) To store map data
- B) For complex navigation behaviors and decision making
- C) To control robot hardware
- D) To generate sensor data

**Answer: B**

### Question 9
What is the Linear Inverted Pendulum Model (LIPM) used for in humanoid navigation?
- A) To control robot wheels
- B) As a simplified balance model for bipedal locomotion
- C) To generate maps
- D) To process sensor data

**Answer: B**

### Question 10
What is the main advantage of using a 3D costmap for humanoid navigation?
- A) It reduces computational requirements
- B) It can account for step height and climbing capabilities
- C) It eliminates the need for sensors
- D) It makes the robot move faster

**Answer: B**

### Question 11
Which Nav2 component is responsible for generating optimal paths from start to goal?
- A) Local Planner
- B) Global Planner
- C) Controller
- D) Costmap

**Answer: B**

### Question 12
What special consideration is needed for crowd navigation with humanoid robots?
- A) Moving as fast as possible
- B) Respecting human personal space and social norms
- C) Avoiding all humans completely
- D) Making loud noises to warn people

**Answer: B**
