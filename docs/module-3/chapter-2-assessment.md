# Chapter 2 Assessment: Perception with Isaac Sim & Isaac ROS

## Assessment Questions

### Question 1
What are the benefits of photorealistic simulation for robotics perception?
- A) Reduced computational requirements
- B) High-fidelity testing environments that closely match real-world conditions for training perception systems
- C) Faster robot operation
- D) Elimination of the need for sensors

**Answer: B**

### Question 2
How does synthetic data generation address real-world data limitations?
- A) It creates more privacy concerns
- B) It provides unlimited data with perfect annotations and controlled conditions
- C) It reduces the quality of training
- D) It makes data collection more expensive

**Answer: B**

### Question 3
What are the key components of a VSLAM system?
- A) Only feature extraction
- B) Feature extraction, pose estimation, map building, and loop closure detection
- C) Only mapping
- D) Only localization

**Answer: B**

### Question 4
How does sensor fusion improve localization accuracy?
- A) By using only one type of sensor
- B) By combining data from multiple sensors to reduce uncertainty
- C) By increasing computational load
- D) By reducing the need for algorithms

**Answer: B**

### Question 5
Which of the following is NOT a type of synthetic data that can be generated in Isaac Sim?
- A) RGB images
- B) Depth maps
- C) Semantic segmentation
- D) Robot physical parts

**Answer: D**

### Question 6
What does EKF stand for in the context of sensor-based localization?
- A) Extended Kalman Filter
- B) Enhanced Kinematic Framework
- C) Efficient Knowledge Formation
- D) Elementary Kinetic Force

**Answer: A**

### Question 7
What advantage does synthetic data offer over real-world data for perception training?
- A) It's always more accurate
- B) It provides unlimited data without privacy concerns and with perfect ground truth annotations
- C) It requires no computational resources
- D) It eliminates the need for sim-to-real transfer

**Answer: B**

### Question 8
What does LiDAR stand for?
- A) Light Detection and Ranging
- B) Laser Distance and Range
- C) Linear Detection and Ranging
- D) Light Direction and Resolution

**Answer: A**

### Question 9
What is domain randomization used for in Isaac Sim?
- A) To reduce computational requirements
- B) To train perception models that are robust to domain shift
- C) To make simulations more expensive
- D) To eliminate the need for sensors

**Answer: B**

### Question 10
Which Isaac ROS package provides GPU-accelerated visual-inertial SLAM?
- A) isaac_ros_detectnet
- B) isaac_ros_visual_slam
- C) isaac_ros_pose_estimators
- D) isaac_ros_computer_vision

**Answer: B**

### Question 11
What is the main challenge with monocular cameras in VSLAM systems?
- A) They are too expensive
- B) Scale ambiguity - difficulty determining absolute scale
- C) They cannot capture color
- D) They have too wide a field of view

**Answer: B**

### Question 12
What is the purpose of loop closure detection in VSLAM?
- A) To close physical loops in the environment
- B) To recognize previously visited locations and correct drift
- C) To reduce computational load
- D) To eliminate the need for feature matching

**Answer: B**
